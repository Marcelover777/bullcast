# UI Styles Catalog — 67 Estilos

## Como usar
Consulte este catalogo na **Fase 1 do vibe-design** para identificar o estilo correto
e obter vocabulario visual especifico. Combine estilos quando necessario (ex: Editorial + Glassmorphism).

## Como combinar estilos

```
Editorial (A02) + Glassmorphism (V01) = revista de luxo digital premium
Brutalism (A01) + Kinetic Type (M02) = agencia criativa disruptiva
Japones (C01) + Minimal Extremo (A07) = portfolio artista nivel Awwwards
Dark Premium (V06) + Particle System (M05) = SaaS tech de alto impacto
Earth Tones (V13) + Hand-crafted (O03) = marca organica artesanal
```

## Regras de selecao

1. **Publico jovem (18-30)**: Y2K, Neon, Gradient Mesh, Vaporwave
2. **Corporativo/Enterprise**: Swiss, Material, Flat + Dark Premium
3. **Luxo**: Art Deco, Editorial, Minimal Extremo, Monochrome
4. **Startups Tech**: Dark Premium, Glassmorphism, Particle Systems
5. **Wellness/Saude**: Biophilic, Organic, Scandinavian, Slow Design
6. **E-commerce Fashion**: Editorial, Monochrome, Collage, Typographic
7. **Gaming/Crypto**: Cyberpunk, Neon Glow, Holographic, Y2K
