# app-dev

## Project Overview
Full-stack application development workspace with React 19, Node.js backend, and cloud deployment.

## Tech Stack
- Frontend: React 19, Tailwind v4, TypeScript
- Backend: Node.js, REST/GraphQL APIs
- Database: PostgreSQL with Prisma ORM
- Auth: JWT / OAuth
- Testing: Vitest (unit), Playwright (E2E)
- CI/CD: Docker, GitHub Actions, Vercel
- Observability: OpenTelemetry, Langfuse
